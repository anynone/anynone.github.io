centos/kylin(yum)安装elasticsearch6.3.1

发表于 2021-09-27 更新于 2024-12-14

安装包下载 (包含ik分词器)

  • 链接: https://pan.baidu.com/s/16hZCo46eHkuqW1_HMTj3Dw
  • 提取码: ekkr

安装路径

  • elasticsearch ： /opt/elasticsearch-6.3.1
  • ip地址: 192.168.56.101

解压,建目录,创建用户,修改权限

安装ik分词器

改elasticsearch配置

  • 配置文件: /opt/elasticsearch-6.3.1/config/elasticsearch.yml

    修改系统配置文件

  • 配置文件: /etc/security/limits.conf 增加配置
  • 配置文件: /etc/sysctl.conf 增加配置 执行sysctl -p并且重新登陆使其生效

    启动elasticsearch

    因为不能使用root启动，这里su到elastic用户启动服务

    解压kibana，测试

    相关端口

    端口 说明
    9200 elasticseach服务端口, rest接口
    9300 elasticsearch集群通信端口
    5601 kibana管理界面监听端口

附加开启端口命令

  • iptables -I INPUT -p tcp –dport 9200 -j ACCEPT
  • iptables -I INPUT -p tcp –dport 9300 -j ACCEPT
  • iptables -I INPUT -p tcp –dport 5601 -j ACCEPT

iptables持久化可以搜索下看看，如果没有，说明还没写

systemctl 服务化

这是service文件，复制到相应系统目录

系统配置

  1. 新建用户

  2. 修改打开文件和进程数内存锁定大小等 /etc/security/limits.conf

  3. 修改申请的内存权限大小 /etc/sysctl.conf增加配置

  4. 新建配置/etc/sysconfig/elasticsearch

  5. systemctl start elasticsearch 启动即可
